What is Emmet?
=> Emmet is an development tool that helps you quickly write repetitive structures 
like lists, tables, or ordered elements with minimal typing. It is more like a shorthand 
that translates to multiple lines of HTML or a CSS attribute with simple syntax.

Difference between a Library and framework?
=> A framework is a set of pre-written code that provides a structure for developing the applications. 
A library, on the other hand, is a collection of pre-written code that can be used to perform specific tasks in application where we needed.for eg: jQuery,
React, ReactDOM

What is CDN? why do we use it?
=> A CDN is content delivery network of geographically dispersed servers which delivers cached content 
from websites to end users based on the geographic locations of the user.

Why is React known as React?
=> t was developed by Facebook, and the name “React” was chosen because it is meant to help developers build user interfaces that are fast 
and responsive, or “reactive.” The library was designed to “react” to changes in data.

What is crossorigin in the script tag?
=> The crossorigin attribute on an element specifies that CORS is supported when fetching files from a 
third party server or domain. CORS is a standard mechanism used to retrieve files from other domains. 
Elements that accept this attribute include <img>, <link>, <script>, <audio>, and <video>
The purpose of crossorigin is used to share the resources from one domain to another domain.
It is used to handle the CORS reguest. CORS(cross origin resource sharing.)
This is the process by which one webpage requests to another domain for fetching resources like audio video etc 
with out leaking information.

What is the difference between React and ReactDOM?
=> React is a javascript library for building user Interfaces. ReactDOM is a javascript library that allows react to interacts with DOM.(renders UI in the browser) using render method

What is the difference between react.development.js and react.production.js files via CDN?
=> react.development.js is a javascript library that contains unminified version of the react code and react.production.js
contains the minified version of the react code

What are async and defer?
=> With async (asynchronous) attribute, HTML parsing and script fetching happens in parallel. Once the scripts are available then HTML parsing is paused and 
scripts are executed. Once the execution is complete, HTML parsing continues.
=> With defer attribute script files are downloaded in parallel while the HTML parsing. 
But the execution is deferred means the downloaded scripts are executed only when the browser finishes its HTML parsing.
